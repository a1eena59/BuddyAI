# BuddyAI
BuddyAI is an AI-assisted team-matching platform for university students. It enables students to discover hackathon teammates and competition partners by matching skills, interests, and goals through a chatbot-driven experience.

The platform focuses on intra-campus collaboration and is built to scale toward a smart campus ecosystem using Next.js, Supabase, and modular AI-ready architecture.
